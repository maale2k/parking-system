# Parking Lot & Permit Management System

## Description
This is a Python-based system that manages:
- Parking spaces
- Vehicle entry/exit
- Permit management
- User roles (Admin, Staff, Officer, Owner)

## Features
- File handling using .txt
- Menu-driven system
- Role-based access
- Validation and error handling

## Technologies
- Python
- Text Files
